# [v1.1.0]

Added Scandinavian alphabets. This package now exports the following alphabets:

| Name                      | Export name      |
|---------------------------|------------------|
| Danish                    | `danish`         |
| Faroese                   | `faroese`        |
| Greek (modern)            | `greek`          |
| Greek (ancient/polytonic) | `greekPolytonic` |
| Icelandic                 | `icelandic`      |
| Latin                     | `latin`          |
| Norwegian                 | `norwegian`      |
| Swedish                   | `swedish`        |
